My Experience

Intern at TIH-IoT Chanakya (IIT Bombay)

During my internship at TIH-IoT Chanakya, IIT Bombay, I was involved in the development of innovative control systems designed to enhance accessibility and mobility for individuals with disabilities. I worked on integrating speech and mind-controlled systems into wheelchair technology, which significantly improved the user experience. My primary focus was on developing a speech control system using Arduino Uno microcontrollers and HC05 Bluetooth modules, which translated user commands into actionable digital signals. I also contributed to the development of a mind-controlled wheelchair system, using L293D motor drivers to translate cognitive commands into precise movements. This project not only honed my programming skills but also deepened my understanding of hardware integration and user-centric design. Working closely with a team of skilled professionals, I was able to contribute to creating a wheelchair that was not only technologically advanced but also intuitive and adaptable to the varied needs of its users.

Intern at MyEdMaster

At MyEdMaster, I played a crucial role in developing and enhancing the SAT gaming platform. My primary task was to analyze user performance data across Math and English categories. I collected, processed, and cleaned the data using Pandas, ensuring its readiness for advanced analysis. This involved handling missing values, normalizing data, and preparing it for statistical methods that ensured reliability and accuracy. I created visualizations with Matplotlib and Seaborn, including histograms and pie charts, to uncover trends and patterns that informed project decisions. I also explored the application of machine learning models, such as logistic regression, to predict user performance and provide feedback for improvement. Additionally, I managed and queried large datasets using SQL, designing complex queries to extract relevant insights. My experience with MySQL and DBeaver enhanced my data engineering skills, particularly in managing and preprocessing large datasets. Beyond data analysis, I contributed to the development of a robust API layer that facilitated data retrieval and manipulation. This API was integral to the platform's functionality and my work in this area gave me a deeper understanding of how data science integrates with software development. As the project evolved, I gained experience in frontend development, designing analysis pages using Django and React. I also enhanced my web design skills using CSS and JavaScript, and contributed to designing webpages for a multiplayer gaming feature within the SAT platform using Unity.
